# Правоугаоник

- Дијагонале су једнаке.
- Правоугаоник је тетиван. Дијагонала је пречник описане кружнице.
- Симетрале страница су осе симетрије правоугаоника. Правоугаоник има две осе симетрије.

```mermaid
graph LR;
    PT["d² = a²+b²"] --> D{d};
    A{a} --> PT;
    B{b} --> PT;
    FO[O = 2a+2b] --> O{O};
    A --> FO;
    B --> FO;
    FP[P = ab] --> P{P};
    A --> FP;
    B --> FP;
```
