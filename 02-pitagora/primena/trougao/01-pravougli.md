# Правоугли троугао

- Најдужа страница се зове хипотенуза, а преостале две странице су катете.
- Оштри углови правоуглог троугла су комплементни.
- Пречник описане кружнице је хипотенуза, а њено средиште је центар.

```mermaid
graph LR;
    PT[c^2=a^2+b^2] --> C{c};
    A{a} --> PT;
    B{b} --> PT;
    FO[O=a+b+c] --> O{O};
    A --> FO;
    B --> FO;
    C --> FO;
    FP[P=ab/2] --> P{P};
    A --> FP;
    B --> FP;
```
